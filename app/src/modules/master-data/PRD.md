# 기초자료 PRD

## 1. 개요
- **모듈명**: 기초자료
- **목적**: EMR 운영에 필요한 마스터 데이터 관리 (약품, 질병코드, 검사, 수가 등)
- **사용자**: 관리자

## 2. 사용자 시나리오

### 시나리오 1: 약품 코드 관리
```
1. 관리자가 기초자료 > 약품 관리에 진입한다
2. 약품을 검색하여 용량, 가격, DUR 정보를 확인/수정한다
3. 신규 약품을 등록한다
```

## 3. 화면 구성

### 3.1 기초자료 메인
- **카드 모듈**:
  - 카테고리 메뉴 (약품/질병코드/검사/수가/진료과/의사)
  - 데이터 테이블 (검색/필터/페이징)
  - 상세 편집 (모달 또는 사이드패널)
  - 일괄 업로드 (엑셀)

## 4. 마스터 데이터 종류
| 데이터 | 설명 |
|--------|------|
| 약품 마스터 | 약품코드, 약품명, 성분, 용량, 가격, DUR 정보 |
| 질병코드 (KCD) | 질병분류코드, 한글명, 영문명 |
| 검사 마스터 | 검사코드, 검사명, 참고치, 가격 |
| 수가 마스터 | 수가코드, 항목명, 보험 점수, 비급여 가격 |
| 진료과 | 과 코드, 과 이름 |
| 의사 | 의사 ID, 이름, 진료과, 면허번호 |
| 묶음처방 | 세트명, 포함 약품 목록, 공유 여부 |

## 5. 연관 모듈
- 진료: 약품/질병코드 검색 시 참조
- 수납: 수가 마스터 참조
- 서식: 서식 템플릿 관리
