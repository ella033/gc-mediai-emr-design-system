# 통계 PRD

## 1. 개요
- **모듈명**: 통계
- **목적**: 진료/매출/환자 데이터 기반 통계 분석, 차트 시각화
- **사용자**: 관리자, 의사, 원장

## 2. 사용자 시나리오

### 시나리오 1: 월간 매출 확인
```
1. 관리자가 통계 화면에서 기간을 선택한다 (이번 달)
2. 매출 추이 라인차트, 진료과별 매출 비교, 보험 유형별 비율을 확인한다
3. 상세 데이터를 엑셀로 다운로드한다
```

## 3. 화면 구성

### 3.1 통계 대시보드
- **카드 모듈**:
  - 기간별 매출 추이 (라인차트)
  - 진료과별 매출 비교 (바차트)
  - 환자 수 추이 (라인차트)
  - 보험 유형별 비율 (도넛차트)
  - 질병 코드별 진료 건수 (TOP 10 테이블)
  - 의사별 진료 건수 비교

## 4. 데이터 명세
- 기간 필터: 일/주/월/분기/연
- 진료과 필터
- 의사 필터
- 보험 유형 필터

## 5. 연관 모듈
- 수납: 매출 데이터
- 접수: 환자 수 데이터
- 진료: 진료 건수, 질병 코드 데이터
