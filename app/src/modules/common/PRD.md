# 공통 레이아웃 PRD

> 모든 모듈에 공통으로 적용되는 LNB(Left Navigation Bar), Topbar, StatusBar 정의입니다.
> 이 레이아웃은 모듈과 무관하게 항상 고정으로 표시됩니다.

---

## 1. 개요

| 항목 | 내용 |
|------|------|
| **목적** | EMR 전체에서 일관된 네비게이션과 상태 정보 제공 |
| **영향 범위** | 홈, 접수, 진료, 수납, 예약, 통계, 서식, 기초자료, 환경설정 — 전체 9개 모듈 |
| **핵심 원칙** | ① 어디서든 모듈 이동 가능 ② 현재 위치 인지 ③ 사용자 정보 접근 ④ 빠른 설정 |

---

## 2. 전체 레이아웃 구조

```
┌──┬──────────────────────────────────────────────────┐
│  │  Topbar (56px, 고정)                              │
│  ├──────────────────────────────────────────────────┤
│L │                                                  │
│N │              모듈별 콘텐츠 영역                     │
│B │              (접수/진료/수납/예약/...)               │
│  │                                                  │
│  │                                                  │
│  ├──────────────────────────────────────────────────┤
│  │  StatusBar (32px, 고정)                            │
└──┴──────────────────────────────────────────────────┘

LNB: 접힌 상태 56px (아이콘만) / 펼친 상태 200px (아이콘+텍스트)
```

---

## 3. LNB (Left Navigation Bar)

### 3.1 상태

| 상태 | 너비 | 표시 내용 |
|------|------|----------|
| **접힌 상태 (기본)** | 56px | 아이콘만 + 호버 시 툴팁 |
| **펼친 상태** | 200px | 아이콘 + 메뉴 텍스트 |

### 3.2 메뉴 항목

| 순서 | 메뉴명 | 아이콘 | 경로 | 설명 |
|------|--------|--------|------|------|
| 1 | 홈 | `chart` | /home | 대시보드 |
| 2 | 접수 | `clipboard` | /reception | 환자 접수 |
| 3 | 진료 | `stethoscope` | /chart | 진료 화면 |
| 4 | 수납 | `file-text` | /billing | 수납 처리 |
| 5 | 예약 | `calendar` | /appointment | 예약 관리 |
| 6 | 통계 | `chart` | /statistics | 통계 |
| 7 | 서식 | `printer` | /forms | 서식 관리 |
| 8 | 기초자료 | `settings` | /master-data | 마스터 데이터 |
| - | 구분선 | - | - | - |
| 9 | 환경설정 | `settings` | /settings | 시스템 설정 |

### 3.3 인터랙션

| 액션 | 결과 |
|------|------|
| 메뉴 아이콘 클릭 | 해당 모듈로 이동 |
| LNB 하단 접기/펼치기 버튼 | LNB 너비 토글 (56px ↔ 200px) |
| 접힌 상태에서 아이콘 호버 | 툴팁으로 메뉴명 표시 |
| 현재 모듈 | 아이콘/텍스트 Primary 색상 + 좌측 인디케이터 바 (3px) |

### 3.4 디자인

| 토큰 | Light | Dark |
|------|-------|------|
| `--lnb-bg` | `var(--gray-900)` | `var(--gray-950)` |
| `--lnb-text` | `var(--gray-400)` | `var(--gray-500)` |
| `--lnb-text-active` | `#FFFFFF` | `#FFFFFF` |
| `--lnb-icon-active` | `#3B82F6` | `#60A5FA` |
| `--lnb-indicator` | `#3B82F6` | `#60A5FA` |
| `--lnb-hover-bg` | `rgba(255,255,255,0.06)` | `rgba(255,255,255,0.04)` |
| `--lnb-collapsed-width` | `56px` | `56px` |
| `--lnb-expanded-width` | `200px` | `200px` |

## 4. Topbar

### 4.1 구조

```
┌────────────────────────────────────────────────────────┐
│  [모듈 타이틀]          [검색] [알림] [설정] [🌙] [프로필]│
└────────────────────────────────────────────────────────┘
```

### 4.2 요소

| 위치 | 요소 | 아이콘 | 설명 |
|------|------|--------|------|
| 좌측 | 모듈 타이틀 | - | 현재 모듈명 (16px Bold) |
| 우측 | 글로벌 검색 | `search` | 환자/약품/질병 통합 검색 |
| 우측 | 알림 | `bell` | 미읽음 카운트 뱃지 표시 |
| 우측 | 설정 | `settings` | 빠른 설정 |
| 우측 | 테마 토글 | `sun` / `moon` | Light ↔ Dark |
| 우측 | 프로필 | 아바타 (이니셜) | 클릭 시 프로필 메뉴 드롭다운 |

### 4.3 프로필 메뉴 드롭다운

| 항목 | 아이콘 | 설명 |
|------|--------|------|
| 사용자명 + 역할 | `user` | 오윤경 (의사) |
| 내 정보 수정 | `edit` | 개인 설정 |
| 레이아웃 프리셋 | `bookmark` | 저장된 레이아웃 |
| 구분선 | - | |
| 로그아웃 | `lock` | 로그아웃 |

### 4.4 디자인

| 토큰 | Light | Dark |
|------|-------|------|
| `--topbar-bg` | `var(--gray-0)` | `var(--gray-900)` |
| `--topbar-border` | `var(--gray-200)` | `var(--gray-800)` |
| `--topbar-text` | `var(--gray-900)` | `var(--gray-50)` |
| `--topbar-height` | `56px` | `56px` |

## 5. StatusBar

### 5.1 구조

```
┌──────────────────────────────────────────────────────┐
│  접속: 오윤경 의사 │ 대기 환자: 3명 │ 자동저장: 14:42  │
└──────────────────────────────────────────────────────┘
```

### 5.2 요소

| 위치 | 내용 | 갱신 |
|------|------|------|
| 좌측 | 접속 사용자명 + 역할 | 로그인 시 |
| 중앙 | 모듈별 요약 (진료: 대기 n명 / 수납: 미수납 n건 등) | 실시간 |
| 우측 | 마지막 자동 저장 시간 | 저장 시 |

### 5.3 디자인

| 토큰 | Light | Dark |
|------|-------|------|
| `--statusbar-bg` | `var(--gray-50)` | `var(--gray-950)` |
| `--statusbar-text` | `var(--gray-500)` | `var(--gray-500)` |
| `--statusbar-border` | `var(--gray-200)` | `var(--gray-800)` |
| `--statusbar-height` | `32px` | `32px` |
