# 홈 (Dashboard) PRD

## 1. 개요
- **모듈명**: 홈
- **목적**: EMR 로그인 후 첫 화면. 오늘의 진료 현황, 대기 환자, 알림을 한눈에 파악
- **사용자**: 의사, 간호사, 원무 (역할별 위젯 다름)

## 2. 사용자 시나리오

### 시나리오 1: 오전 출근 시
```
1. 의사가 EMR에 로그인한다
2. 홈 대시보드에서 오늘 예약 환자 수, 대기 중 환자, 어제 미완료 차트를 확인한다
3. 대기 환자 목록에서 첫 환자를 클릭하여 진료 화면으로 이동한다
```

### 시나리오 2: 알림 확인
```
1. 검사 결과 도착 알림을 확인한다
2. 알림을 클릭하여 해당 환자의 검사 결과로 바로 이동한다
```

## 3. 화면 구성

### 3.1 대시보드
- **진입 경로**: LNB > 홈
- **레이아웃**: 위젯 그리드 (카드 기반)
- **카드 모듈**:
  - 오늘 현황 요약 (접수/진료중/완료/예약 건수)
  - 대기 환자 리스트 (실시간)
  - 최근 알림
  - 미완료 차트 목록
  - 오늘 예약 스케줄
  - 공지사항

## 4. 데이터 명세

### 4.1 오늘 현황
| 필드명 | 타입 | 설명 |
|--------|------|------|
| totalReception | number | 오늘 접수 환자 수 |
| inProgress | number | 진료중 환자 수 |
| completed | number | 진료 완료 수 |
| waiting | number | 대기 환자 수 |
| todayAppointment | number | 오늘 예약 건수 |

## 5. 연관 모듈
- 접수: 대기 환자 데이터
- 진료: 미완료 차트
- 예약: 오늘 예약 목록
- 수납: 미수납 건수
