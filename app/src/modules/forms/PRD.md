# 서식 PRD

## 1. 개요
- **모듈명**: 서식
- **목적**: 진단서, 소견서, 동의서, 처방전 등 의료 문서 양식 관리 및 출력
- **사용자**: 의사, 간호사, 원무

## 2. 사용자 시나리오

### 시나리오 1: 진단서 발급
```
1. 의사가 서식 화면에서 '진단서'를 선택한다
2. 환자를 선택하면 기본정보가 자동 입력된다
3. 진단명, 소견 등을 추가 입력한다
4. 미리보기 확인 후 출력한다
```

## 3. 화면 구성

### 3.1 서식 메인
- **카드 모듈**:
  - 서식 목록 (카테고리별: 진단서/소견서/동의서/의뢰서/기타)
  - 서식 편집기 (필드 자동 채움 + 수동 입력)
  - 미리보기 + 인쇄
  - 발급 이력

## 4. 서식 종류
| 서식명 | 설명 |
|--------|------|
| 진단서 | 질병 진단 확인 |
| 소견서 | 진료 소견 |
| 의뢰서 | 타 병원 의뢰 |
| 동의서 | 시술/수술 동의 |
| 처방전 | 원외 처방전 |
| 진료확인서 | 진료 내역 확인 |
| 입퇴원확인서 | 입퇴원 내역 |

## 5. 연관 모듈
- 진료: 환자 정보, 진단 데이터 자동 입력
- 기초자료: 서식 템플릿 관리
