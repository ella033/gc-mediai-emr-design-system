# 환경설정 PRD

## 1. 개요
- **모듈명**: 환경설정
- **목적**: EMR 시스템 설정, 사용자 관리, 권한 관리, 진료실 설정
- **사용자**: 관리자(주), 의사(개인 설정)

## 2. 사용자 시나리오

### 시나리오 1: 개인 설정
```
1. 의사가 환경설정 > 개인 설정에 진입한다
2. 진료 화면 레이아웃 프리셋을 저장/불러오기 한다
3. 자주 쓰는 묶음처방을 관리한다
4. 알림 설정을 변경한다
```

### 시나리오 2: 시스템 관리 (관리자)
```
1. 관리자가 사용자 계정을 추가/수정/삭제한다
2. 역할별 권한을 설정한다 (의사/간호사/원무)
3. 진료실 정보, 운영 시간 등을 설정한다
```

## 3. 화면 구성

### 3.1 환경설정 메인
- **카드 모듈**:
  - 설정 카테고리 메뉴 (개인/시스템/사용자/권한/진료실/알림)
  - 설정 폼 (토글, 입력, 선택)
  - 레이아웃 프리셋 관리
  - 사용자 목록 (테이블)

## 4. 설정 항목
| 카테고리 | 항목 |
|---------|------|
| 개인 설정 | 테마(Light/Dark), 레이아웃 프리셋, 알림 설정, 묶음처방 관리 |
| 시스템 | 병원명, 주소, 운영시간, 로고 |
| 사용자 관리 | 계정 CRUD, 역할 배정, 비밀번호 초기화 |
| 권한 | 역할별 메뉴 접근 권한, 기능별 읽기/쓰기/삭제 권한 |
| 진료실 | 진료실 번호/이름, 담당의 배정, 장비 연동 |
| 알림 | 접수 알림, 검사 결과 알림, 예약 리마인더 설정 |

## 5. 연관 모듈
- 전체 모듈: 권한 설정이 모든 모듈의 접근 제어에 영향
- 진료: 레이아웃 프리셋, 묶음처방
- 홈: 알림 설정
