## 13. 인터랙션 패턴

### 13.1 검색 → 선택 → 반영

```
1. 사용자가 검색 Input에 키워드를 입력한다 (최소 2자)
2. 300ms 디바운스 후 검색 API 호출
3. 드롭다운에 결과 목록이 표시된다 (최대 10건)
4. 항목을 클릭하면 [대상 영역]에 반영된다
5. 드롭다운이 닫힌다
키보드: ↑↓ 이동, Enter 선택, Esc 닫기
```

### 13.2 카드 모듈 드래그/리사이즈

```
1. 카드 헤더를 마우스 다운 → 드래그 시작
2. 파란 가이드라인이 드롭 가능 위치에 표시
3. 마우스 업 → 새 위치에 안착
4. 레이아웃 상태가 자동 저장 (JSON → 서버)

리사이즈:
1. 카드 모서리/변 드래그
2. 최소 크기 제한 적용
3. 주변 카드 자동 재배치
```
