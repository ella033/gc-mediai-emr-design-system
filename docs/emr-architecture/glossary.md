# EMR 도메인 용어 가이드

비의료 배경 입사자를 위한 필수 용어 사전입니다. 한국 의료 IT / EMR 시스템에서 사용되는 핵심 용어를 모듈별로 정리했습니다.

> 출처: HIRA(건강보험심사평가원), NHIS(국민건강보험공단), EMR 인증 포털, KOSTOM, 국내 EMR 매뉴얼 분석

## 사용 안내

- **검색** — 한글 용어, 영문, 약어, 정의 본문 어디든 일치하면 검색됩니다
- **모듈 필터** — 상단 탭으로 EMR 모듈별 분류 보기
- **관련 용어** — 카드 하단의 점선 링크를 누르면 해당 용어로 즉시 검색
- **역할 배지** — `의사` `간호사` `원무과` `전체` 로 주 사용자 표시
