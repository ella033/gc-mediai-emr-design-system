# 사용자 시나리오 흐름

실제 진료 워크플로우를 단계별로 따라가며 EMR 모듈 간 데이터 흐름을 이해합니다. 11개 모듈 · 45개 시나리오 · 200+ 단계 · reads/writes 데이터 매핑 포함.

## 사용 안내

- **전체 흐름** — 모듈 간 연결 관계를 시각적으로 보여줍니다. 모듈을 클릭하면 들어오고 나가는 흐름과 시나리오 목록이 표시됩니다
- **모듈 탭** — 각 모듈의 구체적 시나리오를 단계별 타임라인으로 확인
- **Reads / Writes** — 각 단계에서 시스템이 보여주는 정보(👁)와 사용자가 입력하는 정보(✏️)를 별도 표기
